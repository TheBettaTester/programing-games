# Jogos na Steam sobre Programação 🎮💻

Programação e games andam de mãos dadas.
Por isso, criei esta lista de jogos da Steam onde o coding é o protagonista.
Esses jogos podem servir como uma introdução divertida ao mundo da programação. ⭐

---

## Games 🎮

### Human Resource Machine
https://store.steampowered.com/app/375820/Human_Resource_Machine/
Um jogo de puzzle onde você programa pequenos "funcionários de escritório" usando lógica e comandos simples para resolver tarefas.

🔗 Steam:  
 

### 7 Billion Humans
Sequência de Human Resource Machine. Aqui você programa vários trabalhadores ao mesmo tempo usando conceitos mais avançados de lógica.

### CodeCombat
Um RPG onde você aprende programação escrevendo código real para controlar seu personagem.

### Lightbot
Um jogo de puzzles focado em lógica de programação, loops e funções.

---

## 💡 Por que jogos?

Jogos podem tornar o aprendizado de programação mais acessível e divertido.  
Eles ajudam a desenvolver:

- lógica de programação  
- resolução de problemas  
- pensamento computacional
